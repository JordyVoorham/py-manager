How to Use PY-Manager

  1. Upload your PYTHON application:
    • Visit the Upload page.
    • Ensure your PYTHON project is zipped and contains a requirements.txt file generated using the terminal command pip freeze > requirements.txt.
    • Ensure your PYTHON PROJECT contains a main script called main.py
    • If you intend to use the logging feature, make sure there's a folder named logs inside your project with a .log file.
    • Press upload.
  2. Manage your applications:
    • Visit the Projects page.
    • Select your PYTHON application from the list.
    • Start or stop your application as needed.
  3. Logging feature:
    • If you've included logging in your PYTHON application, you can view the latest log content on the project details page.
    • The log content will be updated automatically every few seconds to provide real-time information.
